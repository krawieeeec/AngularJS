Films

Show all films

HTTP GET /api/film

Display specific film details

HTTP GET /api/film/{id}

Edit film

HTTP PUT /api/film/{id}

Params

    filmId : name for new list
    name : List of ids connected to list
    originalName : original name of film
    releaseDate : Release date
    genre : Genre of film
    description : Description of film
    cover : link to cover
{ "filmId": 1, "name": "EDITED-Skazani na Shawshank", "originalName": "The Shawshank Redemption", "releaseDate": "16 kwietnia 1995 (Polska) 10 września 1994 (świat)", "genre": "Dramat", "description": "Adaptacja opowiadania Stephena Kinga. Historia niesłusznie skazanego na dożywocie bankiera, który musi przeżyć w brutalnym świecie rządzonym przez strażników i współwięźniów.", "cover": "http://1.fwcdn.pl/po/10/48/1048/6925401.6.jpg" }

Lists

Show all lists

HTTP GET /api/list

Display specific list details

HTTP GET /api/list/{id}

Add new list

HTTP POST /api/list

Params

    ListName : Name for new list
    FilmIds : List of ids connected to list
{ "ListName": "test list", "FilmIds": [47, 12] }

Delete list

HTTP DELETE /api/list/{id}
